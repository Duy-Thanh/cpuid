# Build
You need to install NASM and MinGW64 first and set the environment variables to point to the MinGW `bin` folder and the NASM installation folder. Then, open `cmd.exe` in the source code folder and type `build.bat`. After these steps, you will have a `cpuid.exe` file that you can use immediately!

MinGW64 can be found on (WinLibs official website)[https://winlibs.com] and NASM can be found on (NASM official website)[https://nasm.us]
